Front-End

é a parte de um site ou aplicativo com a qual os usuários interagem diretamente. Ele envolve a criação e a implementação de elementos visuais, como layout, botões, menus, imagens e textos, garantindo que tudo seja apresentado de forma atrativa e funcional em diferentes dispositivos.

Java e Spring Boot

Java é uma linguagem de programação amplamente usada e reconhecida por sua simplicidade, segurança e portabilidade, tenha uma Máquina Virtual Java (JVM). Spring Boot é um framework baseado em Java que facilita o desenvolvimento de aplicações web e microserviços.

Aplicações Web com JavaScript

Aplicações Web com JavaScript são programas que rodam em navegadores da web, utilizando JavaScript como principal linguagem de programação para interação com o usuário, manipulação de elementos da página e comunicação com servidores.

SQL com PostgreSQL

SQL (Structured Query Language) é uma linguagem padrão usada para gerenciar e manipular bancos de dados relacionais. PostgreSQL é um sistema de gerenciamento de banco de dados relacional de código aberto que utiliza SQL como sua linguagem de consulta principal, conhecido por sua robustez.

Programação

É o processo de criar instruções que um computador segue para realizar tarefas específicas. Essas instruções, escritas em uma linguagem de programação, como Python, Java ou C++, permitem que os computadores executem operações matemáticas, manipulem dados, controlem dispositivos de hardware, entre outras funções.

Data Science

É um campo interdisciplinar que utiliza métodos, processos, e sistemas para extrair conhecimento de dados estruturados e não estruturados. Ele combina técnicas de estatística, e aprendizado de máquina para analisar grandes volumes de dados e transformar essas análises em informações.

Marketing Digital

Marketing Digital é o conjunto de estratégias e práticas usadas para promover produtos, serviços ou marcas por meio de plataformas digitais, como sites, redes sociais, motores de busca, e-mails e aplicativos móveis. Diferente do marketing tradicional, o Marketing Digital permite um alcance mais amplo e segmentado.
